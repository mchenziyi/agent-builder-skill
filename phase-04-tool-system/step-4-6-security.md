# Step 4.6 — 设计安全边界

> 敏感工具权限确认 + 参数校验 + 沙箱。

## 风险与防护

| 风险场景 | 防护措施 |
|---|---|
| 用户诱导执行危险工具 | 敏感工具加权限确认（「确认要删除吗？」）|
| 参数注入（SQL 注入等） | 参数校验 + 参数化查询 |
| 文件系统越权 | Agent 限定沙箱目录 |
| 高频 API 调用消耗费用 | 速率限制 + 会话预算上限 |
| 工具返回恶意内容 | 结果长度限制 + 内容过滤 |

## FAQ

**Q: 参数校验该在哪一层做？**
A: 两层都要。Tool Registry 入口做基本类型校验，Handler 内部做业务校验。
